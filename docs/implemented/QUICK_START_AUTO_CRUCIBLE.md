## The 5 Validation Checks

Every significant idea runs through:

1. **VIABILITY** - Can this be done with available resources?
2. **DIFFERENTIATION** - Is this actually new/better?
3. **USER REALITY** - Will you actually use this?
4. **RESOURCE COST** - What's the real investment?
5. **SECOND-ORDER** - Then what? What happens next?

Score 8+/10 ‚Üí presented  
Score 6-7/10 ‚Üí evolved first, then presented  
Score <6/10 ‚Üí rethought or different approach
