## How It Works (3-Second Version)

Every question gets scored:

```
Complexity: Simple=0, Medium=1, Complex=2
Stakes: Low=0, Medium=1, High=2  
Novelty: Standard=0, New=1, Novel=2
Your Signal: "Quick"=0, Neutral=1, "Best"=2
Ambiguity: Clear=0, Maybe=1, Unclear=2

Total Score:
0-2 points ‚Üí SKIP (fast answer)
3-5 points ‚Üí LIGHT (quick check)
6-8 points ‚Üí FULL (complete validation)
9-10 points ‚Üí DEEP (advanced analysis)
```

**You don't see this happening. It's automatic.**
