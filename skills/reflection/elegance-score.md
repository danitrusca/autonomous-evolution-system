# Elegance Score Heuristic (0–10)

+1 for each:

- Pure helper extracted or redundant logic removed

- Naming improved or intent clarified

- Observability improved

- Complexity reduced

- Module boundaries clarified

−1 for each:

- Long function (>40 LOC)

- Nested >3 levels

- Dead code or unclear side effect

- Mixed concerns or inconsistent naming

Normalize total to 0–10.
